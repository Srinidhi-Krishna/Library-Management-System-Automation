# Library Management System

A Python-based application that automates library operations using a **MySQL** database.  
The system is designed to improve efficiency and user experience through automation, reminders, and intuitive UI components.

## Features
- Automated **book issuing and return** management  
- **7-day return reminders** implemented using `tkcalendar`  
- **Real-time clock display** within the application  
- User-friendly **scrollable interface** for smooth navigation  
- **Advanced search functionality** for quick record retrieval  
- **Excel export** feature for reporting and data analysis  
- Secure and structured **database-backed system** using MySQL  

## Tech Stack
- **Programming Language:** Python  
- **Database:** MySQL  
- **GUI:** Tkinter  
- **Calendar Integration:** tkcalendar  
- **Data Handling & Export:** Pandas, Excel  

## Project Use Case
This application can be used by:
- Academic libraries  
- Small institutional libraries  
- Learning purposes for database-driven Python applications
